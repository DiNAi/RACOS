------------------------------------------------------------------------------------------
	        Readme for the RACOS Package
	 		       version 2 Dec., 2015
------------------------------------------------------------------------------------------

The package includes the Java code of the RACOS (Randomized coordinate shrinking classifier for optimization) algorithm. Details can be found in paper [1]. 

[1] Yang Yu, Hong Qian, and Yi-Qi Hu. Derivative-free optimization via classification. In: Proceedings of the 30th AAAI Conference on Artificial Intelligence (AAAI'16), Phoenix, AZ, 2016.
 
You will find examples of usage in the Test/examples.java
